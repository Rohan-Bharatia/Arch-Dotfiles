# Arch-Dotfiles

Configuration files for my Arch Linux setup

## Dependencies

- Kitty
- Hyprland
- Hyprpaper
- Hyprlock
- Hypridle
- Hyprshot
- Waybar
- Wofi
- Neofetch
- Zen Browser
- Pulseaudio
- SwayNC
- Playerctl
- GTK3
- Nerd Fonts

## Usage

Run the ```install_deps.sh``` file, and use the ```cp``` command to copy over the files to the ```~/.config``` directory.
